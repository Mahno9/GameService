# find-object — Найди предмет

Игрок ищет заданные предметы на большом фоновом изображении (аналог «Где Уолдо»). Показываются три верхних предмета из упорядоченной очереди; когда предмет найден, на его место в очереди приходит следующий. Фон можно панорамировать и масштабировать. Баллы зависят от итогового времени прохождения.

## Параметры конфигурации

| Параметр | Тип | Описание |
|---|---|---|
| `backgroundImage` | string (URL) | Фоновое изображение (обязателен) |
| `overlays` | array | Декоративные оверлеи-«шум» |
| `overlays[].image` | string (URL) | Изображение оверлея |
| `overlays[].x`, `.y` | number | Координаты ЦЕНТРА в пиксельном пространстве фона (0..naturalWidth/Height) |
| `overlays[].rotation` | number (по умолч. 0) | Поворот в градусах |
| `overlays[].colorFilter` | string (по умолч. `''`) | CSS-фильтр (`filter`) |
| `overlays[].zIndex` | integer (по умолч. 0) | Порядок слоя |
| `overlays[].scale` | number (по умолч. 1) | Масштаб относительно натурального размера |
| `targets` | array | Упорядоченный список целевых предметов (обязателен). Те же поля, что у оверлеев. Порядок = очередь поиска |
| `scoreThresholds` | array | Пороги баллов по времени |
| `scoreThresholds[].maxSeconds` | integer | Макс. время (сек.) для получения баллов |
| `scoreThresholds[].points` | integer | Количество баллов |
| `sounds.found` | string (URL) | Звук нахождения предмета |
| `sounds.win` | string (URL) | Звук победы |
| `sounds.music` | string (URL) | Фоновая музыка (зацикленная, опционально) |
| `muted` | boolean | Отключить звуки при старте |

Координаты `x`/`y` задаются в пиксельном пространстве фонового изображения и обозначают центр предмета.

## Управление

- **Панорамирование**: перетаскивание одним пальцем по пустому месту.
- **Масштаб**: щипок двумя пальцами (pinch-zoom) или колесо мыши на десктопе. Ограничение: минимум — вписывание фона целиком, максимум — 4×.
- **Перетаскивание предмета**: зажать любой предмет (целевой или декоративный) и тащить. При подъёме предмет чуть увеличивается и получает тень («поднятие»); при отпускании возвращается к нормальному виду («приземление»).
- **Тап по предмету** (≤8 px, ≤300 мс без перетаскивания): если верхний предмет в точке касания — один из трёх активных целевых, он засчитывается. Если целевой предмет перекрыт другим, сначала перетащите верхний в сторону.

## Эффект нахождения

Увеличение масштаба, яркая обводка/вспышка яркости и насыщенности, всплеск искорок (particle burst) и звук нахождения. Затем предмет исчезает, а в очереди появляется следующий.

## Начисление баллов

Берётся наименьший порог `maxSeconds`, при котором `maxSeconds >= итоговое_время`. Если время превысило все пороги — 0 баллов. Таймер запускается с первого взаимодействия.

## Завершение

Когда найдены все целевые предметы — звук победы, подсчёт баллов, плавное затухание и `onComplete({ score, won: true })`. Кнопка ✕ — досрочный выход (`onExit`). Кнопка 🔊/🔇 переключает все звуки игры.
