# sliding-puzzle — Пятнашки

Классическая головоломка «Пятнашки» с несколькими раундами. Изображение нарезается на плитки, которые перемешиваются; игрок собирает картинку, перемещая плитки на пустое место. Баллы зависят от времени сборки каждого раунда.

## Параметры конфигурации

| Параметр | Тип | Описание |
|---|---|---|
| `gridSize` | integer (2–6, по умолчанию 4) | Размер сетки N×N |
| `rounds` | array | Список раундов (обязателен) |
| `rounds[].image` | string (URL) | Изображение раунда |
| `rounds[].shuffleMoves` | integer (≥1, по умолчанию 80) | Количество ходов перемешивания |
| `rounds[].scoreThresholds` | array | Пороги баллов по времени |
| `rounds[].scoreThresholds[].maxSeconds` | integer | Максимальное время (сек.) для получения баллов |
| `rounds[].scoreThresholds[].points` | integer | Количество баллов |
| `sounds.tileMove` | string (URL) | Звук хода плитки |
| `sounds.roundWin` | string (URL) | Звук победы в раунде |
| `sounds.gameWin` | string (URL) | Звук общей победы |
| `muted` | boolean | Отключить звуки при старте |

## Начисление баллов

Для каждого раунда берётся наименьший порог `maxSeconds`, при котором `maxSeconds >= elapsedSeconds`. Итоговый счёт — сумма баллов всех раундов. Если время превысило все пороги — 0 баллов за раунд.

## Управление

Тап/клик по плитке, смежной с пустым местом, перемещает её. Таймер запускается при первом ходе раунда. Кнопка ✕ выходит из игры, кнопка 🔊/🔇 переключает звук.
