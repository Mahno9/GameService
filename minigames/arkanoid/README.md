# arkanoid — Арканоид

Классический арканоид с несколькими уровнями, ценными блоками, бонусами и
визуальными эффектами. Мяч отбивается платформой, разрушает блоки и
зарабатывает баллы; уровень пройден, когда уничтожены все блоки. Игра
заканчивается победой (пройдены все уровни) или поражением (потеряны все жизни).

## Параметры конфигурации

| Параметр | Тип | Описание |
|---|---|---|
| `lives` | integer (≥1, по умолчанию 3) | Жизни на всю игру |
| `levels` | array | Список уровней (обязателен) |
| `levels[].backgroundImage` | string (URL) | Фон за блоками (затемняется) |
| `levels[].totalBlocks` | integer (по умолчанию 40) | Всего блоков на уровне |
| `levels[].valuedBlockTypes` | array | Типы ценных блоков |
| `…[].color` | string (CSS color) | Цвет блока |
| `…[].percent` | number | Доля от общего числа блоков, % |
| `…[].points` | integer | Баллы за разрушение |
| `…[].lifetimeSeconds` | integer (по умолчанию 0) | 0 = бессрочно; >0 — секунды до потери ценности |
| `levels[].ballSpeed` | number (по умолчанию 6) | Базовая скорость мяча |
| `levels[].ballAcceleration` | number (по умолчанию 0.002) | Ускорение мяча |
| `sounds.bounce` | string (URL) | Отбив мяча |
| `sounds.blockBreak` | string (URL) | Разрушение блока |
| `sounds.bonus` | string (URL) | Подбор бонуса |
| `sounds.ballLost` | string (URL) | Потеря мяча |
| `sounds.win` | string (URL) | Победа |
| `sounds.lose` | string (URL) | Поражение |
| `music` | string (URL) | Фоновая музыка (зацикленная) |
| `muted` | boolean | Отключить звуки при старте |

## Поле и физика

Движок (`src/engine.ts`) полностью независим от DOM и работает в
нормализованных координатах: `x ∈ [0,1]`, `y ∈ [0,1]`. Блоки занимают верхнюю
полосу `y ∈ [0.06, 0.50]` в сетке 10 колонок. Отбив от платформы задаёт угол
вылета в диапазоне ±60° от вертикали в зависимости от точки касания.

Ценные блоки с `lifetimeSeconds > 0` через указанное число секунд игрового
времени уровня превращаются в серые «пустые» блоки (стоимость 0).

## Бонусы

Разрушение ценного блока с вероятностью 25% роняет капсулу-бонус. Поймать её
платформой — активировать эффект. Виды бонусов:

| Капсула | Эффект |
|---|---|
| W expand | Расширение платформы (×1.5, 10 с) |
| S shrink | Сужение платформы (×0.6, 10 с) |
| F ballFast | Ускорение мяча (×1.3) |
| L ballSlow | Замедление мяча (×0.75) |
| + extraBall | Дополнительный мяч |
| ✷ laser | Лазерная платформа: тап стреляет (10 с) |
| U shield | Щит: однократный отбив у нижнего края |
| X explosion | Взрыв: уничтожает блоки в радиусе вокруг последнего |
| T slowTime | Замедление времени (×0.5, 10 с) |

Жизнь теряется только когда исчезают **все** мячи. Баллы — сумма стоимостей
всех разбитых ценных блоков по всем уровням.

## Управление

- Перетаскивание пальцем/мышью в любом месте поля двигает платформу (по дельте).
- Наклон устройства (`deviceorientation`, gamma) добавляет скорость платформе —
  работает одновременно с перетаскиванием, без переключателя.
- При активном лазере тап стреляет двумя снарядами с краёв платформы.

Кнопка 🔊/🔇 переключает звук, ✕ — выход. Над полем — легенда «цвет — стоимость»
для текущего уровня.
