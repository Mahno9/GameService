# runner — Прыгалка

Бесконечный раннер. Персонаж автоматически бежит вправо, игрок управляет прыжком и приседанием, уклоняясь от препятствий. Скорость нарастает по настраиваемой кривой. Баллы = пройденная дистанция в метрах.

## Параметры конфигурации

| Параметр | Тип | Описание |
|---|---|---|
| `lives` | integer (≥1, по умолчанию 3) | Количество жизней |
| `speedCurve` | array (x-type: curve) | Кривая роста скорости: `[{distance, speed}, ...]` |
| `characterAsset` | string (URL, x-type: asset:gif) | Ассет персонажа (изображение или GIF) |
| `backgroundLayers` | array | Фоновые слои parallax: `[{image, scrollSpeed}, ...]` |
| `backgroundLayers[].image` | string (URL, x-type: asset:image) | Изображение слоя |
| `backgroundLayers[].scrollSpeed` | number (0–1) | Коэффициент скорости прокрутки относительно скорости бега |
| `obstacleTypes` | array | Типы препятствий (обязателен) |
| `obstacleTypes[].width` | number | Ширина препятствия (px) |
| `obstacleTypes[].height` | number | Высота препятствия (px) |
| `obstacleTypes[].weight` | number | Вес (относительная частота появления) |
| `obstacleTypes[].image` | string (URL, x-type: asset:gif) | Ассет по умолчанию |
| `obstacleTypes[].overcomeImage` | string (URL, x-type: asset:gif) | Ассет при преодолении (опционально) |
| `obstacleTypes[].overcomeSound` | string (URL, x-type: asset:audio) | Звук при преодолении (опционально) |
| `sounds.jump` | string (URL) | Звук прыжка |
| `sounds.land` | string (URL) | Звук приземления |
| `sounds.hit` | string (URL) | Звук удара / потери жизни |
| `sounds.gameOver` | string (URL) | Звук конца игры |
| `music` | string (URL) | Фоновая музыка (зацикленная) |
| `muted` | boolean | Отключить звук при старте |

## Управление

- **Тап** — прыжок (только когда персонаж на земле)
- **Зажатие ≥250 мс** — приседание (снижает хитбокс вдвое)
- Кнопка **🔊/🔇** — переключение звука
- Кнопка **✕** — выход из игры

## Начисление баллов

Баллы = пройденная дистанция (целое число). Игра заканчивается при потере всех жизней; результат всегда `won: true`.

## Физика

- Гравитация: 1800 px/s²
- Импульс прыжка: −700 px/s
- Столкновение: AABB с отступом 10% от каждого края (прощение)
- При столкновении: −1 жизнь + 1.5 с неуязвимости (мигание)
- Скорость читается из `speedCurve` по пройденной дистанции (линейная интерполяция)
